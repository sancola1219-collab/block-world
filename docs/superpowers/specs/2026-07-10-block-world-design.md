# 我的方塊世界 — 設計文件（2026-07-10）

## 目標

Minecraft 風格的第一人稱方塊沙盒遊戲，繁體中文介面。
純前端、零建置、零外部依賴：手寫 WebGL2 體素引擎，雙擊 `index.html` 離線即玩。
「能做多大就做多大」——功能全開，但架構沿用「3D遊戲模板」的鐵律。

## 架構鐵律（沿用模板）

1. 邏輯與渲染分離：狀態只在 60Hz 固定時步 update 內改變，rAF 只渲染。
2. 邏輯模組不碰瀏覽器：`noise / blocks / worldgen / world / mesher / physics / entities / inventory / save` 可在 node 跑測試。
3. 隨機一律 `mulberry32` / 座標雜湊，同一種子產生同一世界。
4. 座標：x 向東、z 向南、y 向上；yaw=0 面向 −z。
5. 觸控＝合成輸入，不另寫邏輯分支。

## 世界

- 區塊 16×16 柱、世界高度 96、海平面 32、y=0 基岩。x/z 無限（依需求生成）。
- 地形：值雜湊 fBm 高度圖＋山脈因子；3D 雜湊挖洞穴；礦脈（煤/鐵/金/鑽石，越深越稀有）。
- 生物群系：草原、森林、沙漠、雪原、高山。樹（橡樹/杉樹）跨區塊決定性生成、仙人掌、花草。
- 光照：天光柱掃描＋鄰柱取樣（洞口有漸層）、面向陰影、四角 AO、玩家身上的「火把光暈」（黑暗中才亮）。
- 日夜循環 10 分鐘：太陽/月亮/星星/程序雲、天空漸層與霧同步。

## 方塊（約 26 種）

草地、泥土、石頭、圓石、石磚、基岩、沙、砂岩、木頭、木板、樹葉(橡/杉)、玻璃、磚塊、
雪草地、仙人掌、水、煤/鐵/金/鑽石礦、螢石(自發光)、紅花、黃花、草叢、羊毛(紅/藍/黃)。
花草用十字面片；水半透明、頂面下陷、可游泳。

## 玩法

- 模式：**創造**（飛行、即挖、全方塊選單）與**生存**（血量、掉落傷害、溺水、依硬度挖掘、掉落物拾取、夜晚殭屍）。
- 挖掘：DDA 射線 5 格、裂痕動畫、選取框。放置不得與自己重疊。
- 生物：豬（白天漫遊）、殭屍（夜晚生成、追擊、日出消失）。盒狀程序建模＋走路擺動。
- 物品欄：9 格快捷欄＋27 格背包；合成：原木→木板、沙→玻璃、圓石→石磚、木板→羊毛? （見 inventory.js 定義）。
- 存檔：localStorage（種子＋方塊差異＋玩家狀態），手動＋每 30 秒自動。
- 音效：WebAudio 合成（挖/放/腳步/受傷/殭屍/環境樂）。
- 觸控：左搖桿移動、右半屏滑動視角、跳/挖/放/升/降按鈕。
