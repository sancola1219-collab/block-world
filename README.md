# 我的方塊世界

Minecraft 風格的 3D 方塊沙盒遊戲。**雙擊 `index.html` 就能玩**（離線可玩、不用安裝任何東西）。

## 特色

- 無限世界：走到哪生成到哪（草原、森林、沙漠、雪原、高山五種生物群系）
- 地下洞穴與礦脈（煤、鐵、金、鑽石，越深越稀有）
- 日夜循環：太陽、月亮、星星、飄動的雲、晨昏霞光
- **創造模式**：飛行（F 或連按兩下空白鍵）、無限方塊、29 種方塊隨你蓋
- **生存模式**：血量、摔傷、溺水、依硬度挖掘、掉落物拾取、合成配方、夜晚的殭屍
- 生物：白天的豬、夜晚追人的殭屍（白天會燒起來）
- 自動存檔（每 30 秒＋暫停時），關掉重開可以「繼續遊戲」
- 手機也能玩：虛擬搖桿＋觸控按鈕

## 操作

| 電腦 | 手機 |
|---|---|
| WASD 移動、滑鼠視角 | 左搖桿移動（推滿奔跑）、右半屏滑動視角 |
| 左鍵挖、右鍵放 | 「挖」「放」按鈕、輕點畫面挖一下 |
| E 物品欄/合成、滾輪換格 | 「物品」按鈕 |
| F 飛行（創造）、Esc 暫停 | 「飛」按鈕 |

## 開發

- 測試：`node --test`
- 本地伺服器（開發用）：`npx http-server -p 8127 .`
- 規範見 `CLAUDE.md`
