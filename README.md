# 我的方塊世界

Minecraft 風格的 3D 方塊沙盒遊戲。

**🎮 線上玩：<https://sancola1219-collab.github.io/block-world/>**（手機、電腦都可以）

或者雙擊 `index.html` 離線玩，不用安裝任何東西。

進度會自動記住（每 30 秒＋暫停時自動存檔，存在瀏覽器裡）——關掉隔天再開，按「繼續遊戲」就接著玩。注意：換瀏覽器或換裝置不會共用存檔。

## 特色

- 無限世界：走到哪生成到哪（草原、森林、沙漠、雪原、高山五種生物群系）
- 地下洞穴與礦脈（煤、鐵、金、鑽石，越深越稀有）
- 日夜循環：太陽、月亮、星星、飄動的雲、晨昏霞光
- **工具系統**：木/石/鐵/鑽石四階的鎬、斧、鏟、劍——對的工具挖得快，劍打怪痛
- **火把**：真的會照亮洞穴和夜晚（煤炭＋木棒合成）
- **TNT**：點一下就點燃，會連鎖爆炸；**床**：晚上睡一覺直接天亮；半磚蓋樓梯
- **食物**：豬排、牛排、蘋果（打樹葉有機率掉），右鍵吃了回血
- 生物：豬、羊（掉羊毛）、牛（掉牛排）、殭屍（白天會燒起來）、**苦力怕**（靠近會爆炸！掉火藥）
- **創造模式**：飛行（F）、無限方塊、**全部物品直接拿**（工具/食物/魔杖/圓盾通通有）
- **生存模式**：血量、摔傷、溺水、30+ 種合成配方（木板→木棒→火把→工具鏈）
- **冒險關卡**：變身主角展開任務冒險！
  - 🧙 **關卡一：魔法學院**——你是小巫師：魔杖發射魔法彈、飛天掃帚翱翔、收集魔法水晶、決戰黑巫師
  - 🛡 **關卡二：超級戰士**——你是超級士兵：回力圓盾、超級跳躍與怪力、擊退機器人軍團、決戰機器人首領
  - 🥷 **關卡三：忍者試煉**——你是見習忍者：手裡劍連發、煙霧彈瞬身 8 格、登上三層寶塔、對決會瞬移的暗影大師
  - 🐉 **關卡四：屠龍勇士**——你是屠龍勇者：勇者聖劍、烈焰弓，對抗繞著你飛、會噴火球的巨龍！
- **更多武器道具**：弓（生存可合成）、金蘋果、治療藥水；投射物有貼心的瞄準輔助
- **更多方塊**：冰塊、黑曜石、鐵/金/鑽石塊、書櫃、野生南瓜與會發光的南瓜燈
- **天氣系統**：晴、多雲、下雨、雷暴會自動變化——下雨天空變暗、視野變窄，雷暴會閃電打雷；雪地群系會下雪
- **特效強化**：挖方塊噴出碎屑、爆炸有飛濺的碎塊、雨雪 3D 粒子、閃電全螢幕亮光
- 自動存檔（每 30 秒＋暫停時），關掉重開可以「繼續遊戲」
- 手機也能玩：虛擬搖桿＋觸控按鈕

## 操作

| 電腦 | 手機 |
|---|---|
| WASD 移動、滑鼠視角 | 左搖桿移動（推滿奔跑）、右半屏滑動視角 |
| 左鍵挖、右鍵放 | 「挖」「放」按鈕、輕點畫面挖一下 |
| E 物品欄/合成、滾輪換格 | 「物品」按鈕 |
| F 飛行（創造）、Esc 暫停 | 「飛」按鈕 |
